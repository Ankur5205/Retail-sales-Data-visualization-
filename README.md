# Ecommerce-Business-Insights-Dashboard

# Overview
This Power BI dashboard analyzes ecommerce sales performance using interactive visuals and KPIs. It helps track sales, profit, customer activity, product categories, and payment methods.


# Key Insights
1. Total Sales Amount: 14K
2. Total Profit: 2435
3. Total Quantity Sold: 157
4. Average Order Value (AOV): 4659


# Dashboard Features
1. Sales analysis by state
2. Monthly profit tracking
3. Category-wise quantity analysis
4. Customer-wise sales analysis
5. Payment mode distribution
6. Profit by sub-category
7. Interactive Quarter and State filters


# Tools Used
1. Power BI Desktop
2. DAX
3. Power Query
4. Excel/CSV Dataset


# Business Insights
1. Clothing category has the highest sales quantity.
2. August generated the highest profit.
3. COD is the most used payment method.
4. Trousers are the most profitable sub-category.



# Conclusion
This dashboard provides a clear and interactive view of ecommerce business performance, helping users make better business decisions through data visualization and analytics.
